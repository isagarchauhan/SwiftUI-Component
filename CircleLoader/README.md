### ✅ `Circle Loader`
A customizable circle loader animation view.

```swift
CircleLoaderView(
  model:
    CircleLoaderModel(
      circleCount: 5,
      circleSize: 30,
      backGroundCircleRadius: 60,
      overlapFactor: 0.8
    )
)
```
### ✅ `Morph Circle Loader`
A customizable circle loader animation view.

```swift
MorphCircleView(
  model:
    MorphCircleModel(
      circleCount: 3,
      circleSize: 100,
      backgroundCircleRadius: 60,
      colors: [.orange, .green, .blue]
    )
)
